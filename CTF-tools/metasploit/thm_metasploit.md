```msfdb init```

```msfconsole```

```db_status```

```search```

```use```

```info```

```connect```

```set, setg```


### View the Variable

```get```

### Change the value of variable to Null

```unset```

### Saving to Grepable file

```spool```

Example: ```spool Downloads/console.log```

```save```
